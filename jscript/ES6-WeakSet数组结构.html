    WeakSet 也会去重
    总结：
    1.成员都是对象；
    2.成员都是弱引用，可以被垃圾回收机制回收，可以用来保存 DOM 节点，不容易造成内存泄漏；
    3.不能遍历，方法有 add、delete、has。
    //声明WeakSet
let weakObj = new WeakSet();
let obj = {aa:'annaiah',bbb:'18'};
// let obj1 = {aa:'annaiah',bbb:'18'}; 两个相同的对象，如果内存空间相同，那么只打印一个 否则两个都答应
//添加
weakObj.add(obj);
// weakObj.add(obj1);
console.log(weakObj)
