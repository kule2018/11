    9、类数组

     1 var obj = {
 2             "2" : "a",
 3             "3" : "b",
 4             "length" : 2,
 5             "push" : Array.prototype.push
 6         }
 7         obj.push("c");
 8         obj.push("d");
 9 
10 //obj为？
    结果：{2: "c", 3: "d", length: 4}
解析：类属性
        属性要为索引，必须要有length，最好有push

        Array.prototype.push = function(target) {
             obj[obj.length] = target;
             obj.length++;
        }
