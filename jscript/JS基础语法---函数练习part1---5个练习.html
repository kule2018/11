    练习1：求两个数字的和:获取任意的两个数字的和

          function getSum(x, y) {
        return x + y;
      }
      console.log(getSum(10, 20));

     

    练习2：求1-100之间所有的数字的和

          function geteverySum() {
        var sum = 0;
        for (var i = 1; i <= 100; i++) {
          sum += i;
        }
        return sum;
      }
      console.log(geteverySum());
