    如何把局部变量变成全局变量? 

    把局部变量给window就可以了
     
     
    函数的自调用---自调用函数
    一次性的函数--声明的同时, 直接调用了
        (function () {
      console.log("函数");
    })();

     

    局部变量
    页面加载后.这个自调用函数的代码就执行完了
        (function (形参) {
      var num = 10;//局部变量
    })(实参);
    console.log(num);

     

    把局部变量给window， 则局部变量变成全局变量
        (function (win) {
      var num = 10;//局部变量
      //js是一门动态类型的语言,对象没有属性,点了就有了
      win.num = num;
    })(window);
    console.log(num);
