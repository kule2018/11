    JS的基本语法要求

    1. 严格区分大小写
    2. 标识符命名要求（与java完全相同）
      首个字符只能是字母、下划线、美元符号
      其他位置只能是字母、下划线、美元符号、数字
    3. 变量的声明
      JavaScript的声明必须要使用var关键字，声明方式有两种。
      声明并且赋值：var 变量名 = 值
      先声明再赋值：
      var 变量名；
      var = "666";
      1. JavaScript中的每行代码都最好加上";",当然不加也不会报错，程序也能执行，但是为了规范，还有良好的习惯最好都加上去。
      2. JavaScript的基本数据类型
    类型标识符 类型说明
    Undefined 表示一个未声明的变量或者已经声明但未赋值，或者不存在的属性。
    String 字符串类型
    Number 数字类型
    Null 表示空类型
    Boolean 布尔类型
    1. JavaScrip变量的数据类型区分
      JS中变量声明时，不会定义变量的数据类型，因此一个变量可以保存任意类型的数据。可以使用typeof 变量来判断变量的数据类型。

    JavaScript代码的使用位置

    1. 在HTML代码中使用
      这种使用方式非常少，主要是规范性，和维护性考虑。
    2. 在script标签中使用
      通过在script标签中定义好JS代码，通过script标签唤醒浏览器中的JS解析器引擎来实现。并且script标签可以放到HTML文件的任意位置。这种使用的方式也不多。
    3. 将JS定义在JS文件中，在HTML代码中导入
      主流的使用方式，方便代码的使用，而且利于维护。

    JavaScript的函数

    函数如同方法，将一些代码进行封装实现功能的重复利用，进而解决代码的重复性。
    JS代码的结构：

    /*
 * function 函数名(参数1,参数2,...){
 *      JS代码;
 *      return 返回的数据;
 * }
 */
    function add(a,b){
        var c=a+b;
        return c;
    }

    通常我们会将JS中需要用到的函数全部存放在一个单独的JS文件中，在HTML页面进行调用，然后通过再调用主体的JS语句的文件。就可以实现JS文件之间的相互调用。必须注意的是HTML语句是由上到下依次执行，所以function 函数JS文件的调用必须在主JS代码文件之前，否则主文件会找不到对应的函数。
