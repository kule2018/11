    本节将学习JavaScript语法的一些重要特性。

    字符集

    JavaScript使用Unicode字符集，因此几乎允许所有字符、标点符号和符号。

    大小写敏感

    JavaScript是一种区分大小写的脚本语言。这意味着函数、变量和关键字是区分大小写的。例如，VAR与var不同，John不等于john。

    字符串

    字符串是JavaScript中的文本。文本内容必须以双引号或单引号括起来。

    示例：字符串

    1
    2
    3
    4
    5
    6
    7
    <script>
         
        "Hello World" //JavaScript string in double quotes
         
        'Hello World' //JavaScript string in single quotes  
     
    </script>

    数

    JavaScript允许您处理任何类型的数字，如整数、浮点数、十六进制等。数字必须不用引号包装。

    整数：1000

    浮动：10.2

    布尔型

    与其他语言一样，JavaScript还包含true或false作为布尔值。

    分号

    JavaScript语句由分号分隔。但是，用分号结束每个语句并不是强制性的，但我们建议这样做。

    例如，JavaScript建议三句不同的语句这样书写：

    one = 1; two=2; three=3;

    空白空间

    JavaScript忽略多个空格和制表符。

    以下语句是相同的。

    示例：JavaScript忽略空白空间

    var one =1;

    var one   =   1;

    var one        =         1;

    注释

    注释可以是单行或者多行，注释的作用是对当前的程序代码做一些说明解释，方便别的程序员读懂代码的逻辑，注释和代码不一样，不会被程序执行。

    单行注释写在双斜杠//的后面，多行注释写在/*和*/之间。

    示例：代码注释

    var one =1; // this is a single line comment

    /* this  

    is multi line

    comment*/

    var two = 2;

    var three = 3;

    关键词

    关键字是JavaScript中的保留词，不能用作变量名或函数名。

    下表列出了JavaScript中使用的一些关键字。

    关键词

    var function if

    else do while

    for switch break

    continue return try

    catch finally debugger

    case class this

    default false true

    in instanceOf typeOf

    new null throw

    void width delete

      需记住的要点：

    1、JavaScript使用Unicode字符集。

    2、JavaScript区分大小写。

    3、JavaScript字符串必须用双引号(“)或单引号(‘)括起来。

    4、JavaScript数字可以存储整数、浮点数、十六进制值，而不用使用引号括起来。

    5、JavaScript布尔值存储真假。

    6、JavaScript中的每个语句都可以使用分号(；)分隔。这不是强制性的，但建议在每个语句的末尾使用分号。

    7、JavaScript忽略多个空白。

    8、多行注释可以包装在/*和*/之间。单行注释可以/开头。

    9、JavaScript关键字是保留的单词。不要将它们用作变量或函数名。
