    多分支IF语句和跳楼现象

    格式：　if（条件表达式1）{

    　　　　　　满足条件表达式1执行的语句

    　　　　}else if（条件表达式2）{

    　　　　　　满足条件表达式2执行的语句

    　　　　}else if（条件表达式3）{

    　　　　　　满足条件表达式3执行的语句

    　　　　}else {

    　　　　　　都i不满足上面条件表达式执行的语句

    　　　　}

    　　　　跳楼现象： if语句的本质就是到达下一楼已暗示不满足之上楼层的条件

     

    sample：用户输入成绩：

     

    　　　　大于85 显示优秀；

     

    　　　　大于70 显示良好；

     

    　　　　大于60 显示及格；

     

    　　　　小于60 显示不及格

     

    代码如下：

    　　　　var num = parseFloat(prompt("请输入您的成绩"));　　//1、接收用户输入的值

    　　　　if (num >=85) {

    　　　　　　alert("优秀，棒棒哒！")　　//2、大于85分 显示优秀

    　　　　}else if (nnum >=70) {

    　　　　　　alert("良好，继续加油")　　　//3、大于70分 显示良好

    　　　　}else if(num >=60) {

    　　　　　　alert("及格，还需努力")　　//4、大于60分 显示及格

    　　　　}else {

    　　　　　　alert("不及格，反省")　　//5、不及格

    　　　　}

    多分支If语句嵌套练习：

    sample：

    　　员工的年终奖：1、工龄小于一年，发月薪的1倍月薪的年终奖。如果月薪大于8000元，发1.2倍。

    　　　　　　　　　2、工龄满一年，发月薪的1.5倍月薪的年终奖，如果月薪大于10000元，发1.7倍。

    　　　　　　　　　3、工龄两年甚至更多，发月薪的3倍月薪的年终奖，如果月薪大于12000元，发3.2倍。

    　　要求：员工输入工龄和月薪，计算出年终奖。

    代码如下：

    　　<script>
    　　var workingYears = parseFloat(prompt("请输入您的工龄"));
    　　var monthlyPay = parseFloat(prompt("请输入您的月薪"));
    　　var annualBonus = 0;
    　　if (workingYears < 1 &&workingYears >=0) {
    　　　　if (monthlyPay > 8000) {
    　　　　　　annualBonus = monthlyPay*1.2;
    　　} else{
    　　　　annualBonus = monthlyPay*1;
    　　　　}
    　　} else if(workingYears < 2 &&workingYears >=1){
    　　　　if (monthlyPay > 10000) {
    　　　　　　annualBonus = monthlyPay*1.7;
    　　　　} else{
    　　　　　　annualBonus = monthlyPay*1.5;
    　　　　　　}
    　　}else if (workingYears > 2) {
    　　　　if (monthlyPay > 12000) {
    　　　　　　annualBonus = monthlyPay*3.2;
    　　　　} else{
    　　　　　　annualBonus = monthlyPay*3;
    　　　　　　}
    　　　　} else{
    　　　　　　alert("没有您的档案");
    　　　　　　}
    　　　　alert("您的年终奖为" + annualBonus + "元")
    </script>
