    Microsoft Visual Studio 2017 找不到 Visual Studio Installer ？

    打开vs2017 ，选择 工具 --> 扩展和更新 --> 联机，搜索：

    Microsoft Visual Studio 2017 Installer Projects

    安装。安装好以后，重启vs2017。

    OK，搞定。
