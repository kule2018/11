    懒加载

    • 什么叫懒加载？就是只有在访问的时候才会进行请求加载，这可以有效提升网站打开的速度
    path: '/memberMes',
name:'memberMes',
加上这行，就可以做到懒加载↓===括号里的路径改成组件的路径，然后就不需要在上面import了
component: r => require.ensure([], () => r(require('@/components/memberMes')), 'demo_1')
},
{
path: '/registry',
name:'registry',
component: r => require.ensure([], () => r(require('@/components/registry')), 'demo_1')
},
