    循环语句分类{

    　　for

    　　while

    　　do ( ) while

    }

    一、for循环语句和for循环的嵌套

    　　for循环格式eg：

    <script>
    var sim = 0;
    for(var i=1;i<=100;i++){
        sum+=i;
}
        console.log(sum)

</script>

     　　表达式“i=1”共运行1次，在循环之前运行；

    　　 表达式“i<=100”是判断能否满足执行循环体的条件，如果满足，循环多少次就执行多少次，不满足时跳出循环体；

    　　 表达式“i++”进行自增操作。

     

    　　for循环的嵌套eg：

    <script>
    for(var i=1;i<=3;i++){
        document.write('外层循环循环第'+i+'次的结果<br/>');
            for(var j=1;j<=2;j++){
                document.write('内层循环循环第'+j+'次的结果<br/>');
            }
        document.write('<hr/>');
</script>

//外层循环一次，内层循环一轮

    二、while循环语句eg（先判断条件，然后执行内容循环）

    <script>
    var i=1;  //初始值
        while(i<=20){     //条件
            document.write(i);
　　　　　　　document.write('<br/>');
          i+=10；
}  
</script>
// 输出值为1；11；

    三、do ( ) while循环语句eg (先执行do内内容，然后判断条件，条件满足继续循环，条件不满足时不再循环)

    do{
	document.write('hello meizi');
   }while(false);

    四、break和continue

    　　a:break:  break为跳出当前循环体，当前循环体结束，执行循环体后面的语句；

     

    　　b：continue:  continue为跳出当前循环过程，继续下一次的循环；
