    • uni-app条件编译：#ifdef #ifndef #endif
    原文出处:本文由博客园博主十行道人提供。
    原文连接:https://www.cnblogs.com/shurun/p/11935451.html

    语法：

    // #ifdef %PLATFORM% 
    这些代码只在该平台编译
// #endif
    #ifdef ：      if defined  仅在某个平台编译
#ifndef ：     if not defined  在除里该平台的其他编译
#endif ：      end if 结束条件编译
%PLATFORM%     需要编译的平台，上面的MP就是各个小程序的意思
    uni-app支持的各个平台
