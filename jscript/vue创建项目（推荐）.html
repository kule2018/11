    　　上一节我们介绍了vue搭建环境的情况，并使用一种方式搭建了一个项目，在这里为大家推荐另一种创建项目的方式。

    　　vue init webpack-simple vuedemo02

    　　cd vuedemo02

    　　cnpm install / npm install

     

    　　npm run dev

     

    　　最后在浏览器输入：localhost:8000即可查看效果
