         前端通过请求获取数据流生成文档文件的方式是一种开发下载文档的方式，那么在前端如何将数据流转成文档呢？代码如下

      const element = window.document.createElement('a'); //创建一个元素
      const blob = new window.Blob([content]); // 创建一个Bolb对象
      element.download = `${fileName}.doc`; // 设置下载文件的名称
      element.href = window.URL.createObjectURL(blob); // 将Blob对象传入，创建一个DOMString
      element.click();  // 触发点击事件
      window.URL.revokeObjectURL(blob); // 释放刚刚创建的URL对象
