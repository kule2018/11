    • 关于Vue中，使用watch同时监听两个值的实现方法
    原文出处:本文由博客园博主jANE2160提供。
    原文连接:https://www.cnblogs.com/jane2160/p/11718561.html

     

    1. 先在computed中，用需要监听的两个值(start、end)定义一个对象(dateRange)

    computed: {
    dateRange () {
      const { start, end } = this
      return {
        start,
        end
      }
    }
  })

     

    2. 然后在watch中监听这个对象(dateRange)

    watch: {
    dateRange (val) {
      console.log(val)
    }
  }
