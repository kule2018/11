    1.安装Node封装模块

    安装Node封装模块很重要，因为开发项目中会用到各种各样的功能，这时就需要去下载开源的模块

    使用npm install <module_name> module_name为模块名称，命令下Node模板到你的开发环境中，并将其放置在node_modules文件夹中，

    例如下载express模块

    npm install express

    运行---cmd--命令运行框进入node.js的node_modules文件下

    输入express --version验证
