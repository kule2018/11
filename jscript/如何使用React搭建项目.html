    1.首先说明node.js、npm、cnpm分别是做什么的？

    • node.js简单的说 Node.js 就是运行在服务端的 JavaScript，安装了node.js默认安装了npm，可以使用npm -v查看是否安装。
    • npm可以理解为远程仓库，如果我们要使用npm需要先把需要的包写到package.json文件中，并执行npm install。下载的包文件都会存到node_modules中。
    • cnpm解决了npm无法下载国外文件问题，在国内与npm仓库进行了同步，频率为10分钟每次，尽量保证包的可用性。

    2.cnpm有哪些问题？

    • cnpm 的仓库只是 npm 仓库的一个拷贝，它不承担 publish 工作，所以你用 cnpm publish 命令会执行失败的
    • 不仅是 publish 会执行失败，其它的需要注册用户(npm adduser)、或者修改 package 状态等命令都无法用 cnpm

    3.如何搭建React开发环境?

    • 安装cnpm， npm install -g cnpm --registry=https://registry.npm.taobao.org，设置镜像地址。
    • cnpm install -g create-react-app   安装全局环境的react创建命令。
    • 进入到工作空间，创建前端项目 create-react-app circle-react-demo。
    • 进入到创建项目，通过npm start启动项目。
