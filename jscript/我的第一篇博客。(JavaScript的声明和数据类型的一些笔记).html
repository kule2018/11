    下面是前端js的变量和数据类型的一些笔记，不是很全请多多包涵。

    1.变量
    变量的声明
    var 变量名
    变量这个容器中放的是数据
    变量的赋值
    变量名 = 数据
    var a;
    a = 123;
    声明 和 赋值 合并
    var b = 123;
    变量的连续声明
    var c = 123,
    d = 456,
    e = 789;
    变量命名规则：
    1.不能以数字开头
    2.不能包含特殊字符( $ _ 不算特殊字符 )
    3.不能包含关键字和保留字(候选关键字)
    4.不能使用中文
    规范：驼峰命名法


    2.数据类型
    基本数据类型 值类型
    number string boolean null undefined
    复杂数据类型 引用类型
    function object array

    数值
    字符串必须加引号 "" 空串 " " 空格字符串
    布尔值 只要两个值 true false
    undefined 未赋值
    null 值为空 一般null用来给变量占位
    所以值类型数据都存放在计算机内存的栈区
    栈区的特点：数据的排列都是有序的
    栈区容量比较小，因此只能存放值类型数据，因为值类型数据的数据量比较小
