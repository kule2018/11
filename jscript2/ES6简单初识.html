    export，import ，export default 使用

    ES6模块主要有两个功能：export和import export用于对外输出本模块（一个文件可以理解为一个模块）变量的接口 import用于在一个模块中加载另一个含有export接口的模块。

    示例

    单个变量 导入

    import { name } from "/.a.js"

     

    单个变量 导出

    export var name="李四";

     

     

    如果导出多个变量写法如下

    var name1="李四";
var name2="张三";
export { name1 ,name2 }

     

    导入多个变量写法如下

    import { name1 , name2 } from "/.a.js"

     

     

    导出函数如下写法

    function add(x,y){
   alert(x*y)
  
}
export { add }

     

    导入函数如下 

    import { add } from "/.a.js"
