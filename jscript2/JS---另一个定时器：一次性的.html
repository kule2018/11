    之前学的定时器：setInterval和清除定时器 clearInterval(定时器id);

        //常用的,反复的执行
    window.setInterval(function () {
      alert("哈哈");
    }, 1000);

    window.clearInterval(定时器id);//清理定时器

     

    另一个定时器-------一次性的定时 （点过就没了，不点就一直占空间）

    1.   参数1:函数
    2.   参数2:时间---1000毫秒---1秒
    3.   返回值:该定时器的id
     
    清除定时器：
    window.setTimeout(函数,时间);

     

    点击这个按钮,停止一次性的定时器

        //页面加载后
    window.onload = function () {
      //一次性的定时器
      var timeId = window.setTimeout(function () {
        alert("您好");
      }, 1000)
      //点击按钮，停止这个一次性的定时器
      document.getElementById("btn").onclick = function () {
        clearTimeout(timeId);
      };
    };
