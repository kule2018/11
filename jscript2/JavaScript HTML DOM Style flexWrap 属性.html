    flexWrap 属性

    flexWrap属性指定flex项是否应该换行。
    注意：如果元素不是flex项，则flexWrap属性不起作用。
    如果必要，使flex换行：
    document.getElementById("main".style.flexWrap = "wrap";

    　

    语法

    返回flexWrap属性：
    object.style.flexWrap
    设置flexWrap属性：
    object.style.flexWrap = "nowrap|wrap|wrap-reverse|initial|inherit"

    　

    属性值

    值描述
    nowrap 默认值。 指定灵活项目不会换行
    wrap 指定在必要时将灵活项换行
    wrap-reverse 指定灵活项目将根据需要以相反顺序换行
