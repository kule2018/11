    关于闭包先了解函数执行环境，作用域链以及变量对象

    在函数调用的时候，会创建一个函数的执行环境，这个执行环境有一个与之对象的变量对象和作用域链

    ExecutionContext = {
    variableObject: { .... },
    this: thisValue,
    Scope: [ // Scope chain
      // 所有变量对象的列表     ]
};
