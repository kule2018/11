    2.多版本安装

    　　-1.卸载已有node
    　　-2.安装nvm
    　　　　-在磁盘创建根目录
    　　　　-在根目录下创建两个子目录（nodejs、nvm)
    　　　　-将下载的nvm.zip解压到nvm目录，运行install.cmd回车，配置生成文件，另存为当前目录
    　　　　　　-root： 修改为当前路径
    　　　　　　-path： 修改为nodejs目录路径
    　　-3.配置nvm和nodejs环境变量
    　　　　-添加NVM_HOME，值为nvm目录路径
    　　　　-添加NVM_SYMLINK,值为nodejs目录路径
    　　-4.配置path环境变量
    　　　　-path值加入;%NVM_HOME%;%NVM_SYMLINK%;
    　　-5.测试
    　　　　-nvm version
    　　-6.安装node.js
    　　　　-nvm list 查看当前安装哪些node版本
    　　　　-nvm install latest 安装最先版本
    　　　　-nvm install 10.15.3 安装指定版本
    　　-7.测试是否成功
    　　　　-nvm list
    　　-8.使用指定的node
    　　　　-nvm use 版本号
