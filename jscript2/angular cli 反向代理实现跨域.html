    2、前端跨域 

    项目目录下新建proxy.config.json文件

     

     

     proxy.config.json

     

    {
  "/api": {  
    "target": "http://localhost",    // php服务路径
    "logLevel": "debug",  
    "changeOrigin": true, 
    "pathRewrite": { 
      "^/api": ""
    }
  }
}
