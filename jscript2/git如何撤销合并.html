    撒销一个合并

    如果你觉得你合并后的状态是一团乱麻，想把当前的修改都放弃，你可以用下面的命令回到合并之前的状态：

    $ git reset --hard HEAD

    或者你已经把合并后的代码提交，但还是想把它们撒销：

    $ git reset --hard ORIG_HEAD

    但是刚才这条命令在某些情况会很危险，如果你把一个已经被另一个分支合并的分支给删了，那么 以后在合并相关的分支时会出错。
