    一、拉远程仓库项目代码

    git branch -r :查看远程分支

    合并分支到本地分支：（当前位于本地分支dev）

    1.git  fetch  origin  dev //下载远程 分支 origin/dev

    2.git log -p  dev..origin/dev   //查看本地分支和远程分支的差异

    3. git merge  origin/dev //合并远程分支dev到本地分支dev
    ---------------------
    二、向远程仓库提交代码

    1.git add .   //提交到暂存区

    2.git commit “注释”  //提交到历史区

    3.git remote add origin https://github.com/tugenhua0707/testgit.git     //远程推送
