    • javascript实现网页倒计时效果
    原文出处:本文由博客园博主dreams7433提供。
    原文连接:https://www.cnblogs.com/phperlinxinlan/p/11271700.html

    一、HTML代码如下：

    <div class="timer" id="timer">
     <span style="color: black;">广告剩余:</span><span class="clock" style="color: red;">60</span><span style="color: black">秒</span>
</div>

    二、JavaScript代码如下：

    <script type="text/javascript">
    var t = 60;
    var time = document.getElementsByClassName("clock")[0];

    function time() {
        t--;
        time.innerHTML = t;
        if(t <= 0) {
            //隐藏该div
            document.getElementById("timer").style.display = "none";
            clearInterval(inter);
        }
    }
    var inter = setInterval("time()", 1000);
</script>
