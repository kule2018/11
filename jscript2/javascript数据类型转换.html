    js中  除了  0  null  undefined fals “” NaN 是假值之外  其他的都是真值

    快速将number类型转换成string类型的方法  在number后面加一个空字符

    var str  = 1 + “”

    console.log(typeof(str)) // String

     

    快速将字符串转换成数字  也可以使用+

    var str = “1”；

    str = +str；

    console.log（typeof(str)）//number

    控制台测试如下：

    let a = "1"
    a
    "1"
    typeof(a)
    "string"
    a=+a
    1
    typeof(a)
    "number"
