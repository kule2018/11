    • vue 动态绑定height以及v-if、v-else的使用
    原文出处:本文由博客园博主林景之提供。

    动态绑定height:

    :style="{height: slideHeight+'rem'}"

slideHeight: 2

    如果需要计算来得到高度，如:

    <p :style="obj">111</p>
    data:
    winHei: screen.availHeight, //屏幕高度
winWid: screen.availWidth, //屏幕宽度
obj:{
    height : ''  
}
    this.obj.height = winHei * 3 +'px';

    v-if、v-else
    例：

    <div v-if="isShowSlide" class="shopSlideImg" style="height:2rem">
    <shop-slide :shopImgsList="shopImgsList"></shop-slide>
</div>
<div v-else="isShowSlide" class="shopSlideImg" style="height:0rem">
    <shop-slide :shopImgsList="shopImgsList"></shop-slide>
</div>
