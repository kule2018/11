    三大主流框架：

    1. Vue.js 是目前最火的一个前端框架，react是最流行的前端框架 (react除了开发网站，还可以开发手机app，Vue语法也是可以用于手机App开发的，需要借助于wexx)。

    2. Vue.js 是前端主流框架之一，和 angular 和 react 一起并称为三大主流框架。

    3. Vue.js 是构建用户界面的框架，只关注视图层(视图层只针对前端页面)，它不仅容易上手，还便于与第三方库 或 既有项目整合（vue 有配置第三方库，可以整合起来做大型项目开发）。

    4. 前端的主要工作有哪些：主要负责 MVC 中的这一层，主要工作就是和界面打交道，来只做前端页面效果。

    为什么学习流行框架？

    1. 企业为了提高开发效率，在企业中，时间就是效率，效率就是金钱。

    2. 提高开发效率的发展历程：原生 js -> Jquery 值类的库 -> 前端模板引擎 -> Angular.js/Vue.js (能够帮助我们减少不必要的 dom 操作，提高渲染效率，他俩提供了一个双向数据绑定，通过框架提供的指令，我们前端程序员只需要关心数据上的业务逻辑，不在关心 DOM 是如何渲染的)

    框架和库的主要区别？

    1.  框架是一套完整的解决方案，对项目的侵入性很大(比如我们用vue写了一个项目，想换angular改这个项目，这就需要从新写了)，项目加载需要更换框架，则需要重新构架整个项目。（提供了我们一套的业务解决方案）

    2.  库(插件)：提供了一个小功能，对项目的侵入性较小，如果某个库无法完成某些需求可以很容易切换到其它库实现需求
