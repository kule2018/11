    • 前端从一只小白到工作半年的心路历程
    原文出处:本文由博客园博主初心，你好吗提供。
    原文连接:https://www.cnblogs.com/LHLVS/p/10859044.html

            提笔又不知从何说起， 毕竟头发还茂盛，离大神就差一头秀发的距离，

    所以此文不讲技术， 只是单纯的总结一下我是怎么入坑的。本人前端打杂的，不喜勿喷。

            17年7月我毕业了， 可我并不知道未来在何方， 我该做什么，大学四年给我的只是一纸证书

    由于家在南方， 所以就选择了深圳这个繁华又孤独的城市， 搜索和专业有关的工作，

    机械地海投简历，焦虑， 想做出改变又不知努力的方向， 我迫切想找到一份工作，

    不论待遇，我只是不想毕业了还在增加父母的负担， 但我依然不知道我该做什么，

    幸运的是， 可能是这一纸证书起了作用吧（某985 电子信息工程专业 ），第一个面试什么问题都没答上来竟然通过了，

    我没多想直接入职了，此时7月14， 先解决温饱要紧，这家企业当时招聘信息上写的是射频测试， 我当时还想

    射频是啥， 能吃吗 ? 哈哈哈， 此后一年的时光证明了太容易得到的果然是个坑， 存粹就是个苦力活

    还两班倒， 不过现在天天坐在椅子上还有点怀念之前做苦力的日子。。。。。我果然犯贱。

    　　回到正题，当时做的射频测试和我们组有个相关的组是处理数据的，他们用的python， 

    我真是脑子一热就开始学了，当时对编程一窍不通， 大学C++还挂过科， 你们自行脑补我有多菜了，

    记得第一本编程书是Head First Python, 写出一个hello world 都高兴半天，然鹅， 动力越来越不足，

    但是还是不想放弃做一个高逼格的互联网人，既然单挑不行，得叫个打野来帮我了， 我开始在各大平台上搜课程，

    各种语言都搜了个遍， 当时想先挑个最简单的入门然后再从长计议，我连PS的课都看了。。。。。。，最终我选择了

    能立马出效果的前端，此时到了18年2月， 过年大家都高高兴兴的， 我也不敢对父母说我的工作状况（当时上班的时间

    学python被领导发现了，从此我呆着就很难受了，但是同事人很好），回到深圳， 开始入坑， 第一本书又是head first Html。。

    结合教学视频学起来还是很快的， html + css 一周撸完 ， 然后撸个淘宝的静态首屏（就是没有任何效果的淘宝网首页）暂时结尾，

    开始js的学习， 前面学概念做不出demo的日子真的难熬啊，作用域链， 闭包，原型链，圣杯继承， 深拷贝这些看了又忘，忘了

    我还不想回去看。。。。。。懒惰是原罪， 懒惰一时爽，一直懒一直爽。。。。。。然后学到dom操作和事件，我感觉等级终于起来

    了那么一点点， 不用被野怪欺负了， 之前在抖音看到的那个点一下按钮消息框满屏幕随机走的效果我还在评论膜拜来着， 

    现在看来，卧槽，老子也能写了。

    　　时间一晃来到18年7月， 我还没辞职， 但是两班倒的状态还有一大波同事离职导致我越来越忙，每天下班累成🐶，越来越不能静下心来学习，

    所以18年7月14号，刚好一年，我提了离职，刚学到h5的新特性，本人性格又有点不自信， 看到招聘需求上写的那些条件，至少精通一门框架，我退缩了， 我选择了回家先把基础打好先， 你没看错，我选择了回家。。。回家后父母也没说我什么，只是劝我别回深圳了，在家考个公务员。。。。。。安安稳稳一辈子。。。。。。还给我例举了数个的别人家的孩子，这更加坚定了我要学好前端的想法，我可以回家， 但不是在20多岁的时候，在家的时间过的很快，时间轴： 18年10月， 我回到了深圳， 第一天满怀希望在拉勾网的投了40份简历， 95%全是不匹配，第二天依旧，有点慌，第三天boss直聘更新简历，立马就有面试了。。。没敢投大厂，心理素质还不强大，最后选择了一家老大是我学长（吃软饭的感觉。。。）

    待遇说不上好，也不差， 但是他们的技术栈是angularjs + php +  mysql  ，全不会。。。关键项目还是要从angularjs 重构 升级到 angular7  ，

    我的妈呀，他们表示可以培养， 我也就这样入坑了。啰嗦了一大堆， 还是想告诉学弟学妹们， 大学不努力 ，以后就只能像我这样了，可我们没有

    重来一次的机会。
