    vue更新换代很快，马上vue都要出3.0了，这是一个巨大的变革，不过今天讲的是vuecli3.0，里面使用的vue仍然是2的，所有可以放心大胆使用。

    Vue CLI 是一个基于 Vue.js 进行快速开发的完整系统,里面所有具体介绍都可以去官网查看到vuecli官网移步

    这里主要是讲一些我实际中使用遇到的一些东西与经验吧。

    我的项目中主要用到这些：

    • vue 2.6.6
    • vuecli 3.x
    • vue-router 3.0.2
    • vue-axios 2.1.4
    • vuex 3.1
    • stylus 0.54
    • es6
    • swiper4
    • 所有接口都是网上第三方提供

    不多说按照官网的安装搭建
    Vue CLI 需要 Node.js 8.9 或更高版本 (推荐 8.11.0+)

    npm install -g @vue/cli
# OR
yarn global add @vue/cli

创建一个项目
vue create hello-world

    一路自己选中或者vue ui用可视化ui页面设置

    之后就会自动生成一套本地项目环境文件，在这里你就可以为所欲为了。。。
