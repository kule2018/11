    • 离开输入框后判断优惠价是否小于原价
    原文出处:本文由博客园博主hanhe提供。
    原文连接:https://www.cnblogs.com/shizong2/p/11062621.html

    原理就是根据jquery的on方法的bluer事件（bluer事件就是当输入框失去鼠标光标）判断两个输入框的值大小。

    首先定义一个原价输入框，设置class为pri
    <input class="pri" type="text" placeholder="请输入原价">

    再来一个优惠价输入框，设置class为npri

    <input class="npri" type="text"  placeholder="请输入优惠价">


    接下来在js，里面写监听事件

    当第一个输入框失去焦点
    $('.pri').on('blur',function(){
        var npri = $(".npri").val()//获取优惠价的值
        var pri = $(this).val()//获取当前输入框的值

    //如果两个输入框都不是空就比较
        if (npri != '' && pri !=''){
            if ($(this).val() <= npri){
                alert('优惠价需小于原价');
                $(this).val('');
            }

        }

    })
     
    $('.npri').on('blur',function(){
        var pri = $(".pri").val()
        if (pri != ''){
            if ($(this).val() >= pri){
                alert('优惠价需小于原价');
                $(this).val('');
            }
        }
    })
