    • 解决incorrect 'only available in ES6' warning (W119) with options `moz true, esversion 6` 报错问题
    原文出处:本文由博客园博主那片星空，那片海提供。
    原文连接:https://www.cnblogs.com/jinxing-tc/p/10822134.html

    很多同学在新建vue项目时，会遇到 incorrect 'only available in ES6' warning (W119) with options `moz: true, esversion: 6` 得报错，原因是装了jshint插件，对ES6语法进行了检查

    下面贴出解决办法

    方法一：在项目下新建一个名为.jshintrc的文件，里面输入

    {
    "esversion": 6
}
    保存即可。

    方法二：修改vs code设置
    点击 file -> preferences -> setting
    你可以看到User Settings 和 Workspace Settings
    修改User Settings会影响到所有项目得设置，修改workspace settings则只会影响当前存在的项目的设置，具体修改哪个根据需求而定。

    这里我贴上修改workspace settings的截图：

    打开settings.json文件，在大括号里面加入

    "jshint.options":{
    　　"esversion":6
    }
    即可。
